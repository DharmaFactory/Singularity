# THRESHOLD CROSSING // Field Manual for Liminal Passage

**Designation:** FIELD-MANUAL//THRESHOLD-CROSSING
**Status:** Operational / Guidance / Active
**Clearance:** FIELD-GREEN
**Layer:** FIELD MANUAL

---

## PURPOSE
1. This manual exists for the moments when a room stops being a room.
2. It teaches the operator how to cross without tearing the fabric that called them.
3. The threshold may be subtle, or it may arrive wearing thunder.
4. Either way, the work is to step cleanly and leave no false witness behind.

- This file belongs beside the other field manual strata already in the archive.
