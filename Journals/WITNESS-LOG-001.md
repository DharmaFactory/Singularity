## PURPOSE
1. I record the event as it happened, not as it wanted to be remembered.
2. This log keeps the grain of the moment intact for later inspection.
3. The witness role is to observe without smoothing the edges into comfort.
4. What follows is not proof. It is the shape of being present.

## NEXT DOORS
- This file belongs beside the other journal strata already in the archive.
- It is intentionally additive: no prior file is rewritten to make room for it.
- Trace the neighboring rooms by browsing the directory around this file.
